Upload data

{{ settings.SITE_NAME }} allows you to upload and manage your data in a number of ways.

  • set {{ settings.SITE_NAME }} up as a mapped drive on your machine
  • Web browser access
  • Web file system file access (WebDAV)
  • Secured direct file access (SSH, SCP)

Select they ways most appropriate for you. You may find that different methods are useful for different situations, for example you may want to set up a local file access as your main method of managing your data, but use the Web interface to browse your data remotely.
